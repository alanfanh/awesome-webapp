### 语言

```text
python3.6 64bit
```

### 依赖

> 可使用"pip install -r requirements.txt"一键安装所有依赖项

````text
flask=1.0.2
aiohttp=2.3.8
jinja2=2.10
aiomysql=0.0.11
````
